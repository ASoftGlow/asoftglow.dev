I was interested in making one of my own to interact with my friends, but there was an issue: I did know any programming languages in which to code it in. Previously, I had mainly been making Scratch projects. Pathetic, I know. I had been wanting to learn a programming language for while. However, this bot idea gave me the inspiration to do so.

So, with minimal Python knowledge, I began coding.
